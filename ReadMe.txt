Folders required for working :
1) input --> image, video
2) output

Libraries required for working :
1) opencv-python
2) Dlib
3) face_recognition
4) moviepy

working :
1) add one video in input-->video folder
2) add one image in input-->image floder
3) run recognize.py
4) output is generated in output folder
